Permutations and Combinations

If you remember your probability, you'll recall that a permutation is a unique ordering of a particular set. The number of possible permutations ordering k elements from a set of n elements is n!/(n-k)!. A combination is the same as a permutation, except that the order doesn't matter. The number of combinations choosing k elements from a set of n elements is n!/{k!(n-k)!}.

Perhaps it's worth noting the banal fact that we have four pieces of information: two definitions, and two equations which are ways to calculate the definitions. That is, the definition is the specification, independent of how the value is calculated.
